# Currículum Vitae - Adrian

Este repositorio contiene la estructura y diseño de mi Currículum Vitae (CV) desarrollado en código web.

## Descripción

El archivo principal, `index.html`, está diseñado para mostrar mi información profesional, habilidades, experiencia laboral y educación de forma estructurada para poder visualizarse en cualquier navegador web.

Este proyecto sirve para:
* **Presentación Personal:** Mostrar mi perfil profesional de manera digital.
* **Fácil Visualización:** Cualquier persona puede ver el CV abriéndolo directamente en su navegador.
* **Control Total:** Mantener el CV en código permite personalizar el diseño al 100% y actualizar los datos en segundos.

## Cómo visualizarlo

Para verlo, simplemente ubica el archivo `index.html` en tu carpeta y haz doble clic sobre él. Se abrirá automáticamente en tu navegador web predeterminado (Chrome, Edge, Firefox, Safari, etc.).

## Edición

Si necesitas actualizar algún dato de contacto o experiencia, solo abre el archivo `index.html` en tu editor de código preferido, modifica el texto y guarda los cambios.
